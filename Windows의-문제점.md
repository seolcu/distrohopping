# Windows의 문제점

- Bloatware가 많음
- 설치 시 마이크로소프트 계정 계속 권유함
- 파일시스템 구분자 역슬래쉬 -> 정규식 혼란
- 프로그램의 설치 과정이 복잡함(GUI를 이용한 설치)
- 성능 떨어짐
- 유료임
- 바이러스에 취약
- 설정의 파편화(제어판과 설정앱의 역할이 겹침)
- 기본적으로 클램쉘 지원 안함
- 기본적으로 개인정보를 너무 많이 캐감
- 수많은 Unix-like 앱들 사용불가
- 개발환경 설정이 너무 귀찮음(Ex. 환경변수 설정 안되면 따로 해줘야함,,,)
- 사용자의 피드백을 듣지 않음
- 업데이트 과정이 매끄럽지 않음(무한로딩 등)
- 업데이트를 강요함
- 스토어 설치가 느리고 최신 버전이 아닌 경우도 있음(Ex. 아두이노 IDE)
- 스토어에 지들 넣고 싶은 거만 넣음
- WSL 설치 에러
- Hyper-V는 Windows Pro 이상만 쓸 수 있음
- TPM, Secure boot를 강제하는 윈도우11때문에 수많은 컴퓨터들이 2025년에 EOL을 맞게 되어버림
- 듀얼부팅시 업데이트하면서 GRUB을 날려버리는 경우도 있음
- Fast Startup을 기본적으로 켜둠
- 기본 문자 인코딩 옵션이 UTF-8이 아님: 코딩할때 불편함
