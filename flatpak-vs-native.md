# Flatpak vs Native

어떤 앱을 Flatpak으로 설치하고, 어떤 앱을 네이티브로 설치해야 하는지.

## Flatpak

- Brave/Firefox: 배포판과 상관없이 코덱 제공됨
- Bottles: 개발자 권장
- 기타 대부분의 프로그램들

## Native

- Steam: Flatpak은 한글 입력(Terraria)과 바로가기 추가가 안됨. 또한 Native에서는 공식 ProtonGE 적용 가능
- VSCode: Flatpak은 터미널이 안됨
- ProtonVPN: 공식적으로 지원됨
- Boxes: USB Passthrough 가능
