# HDR 세팅법

## HDR 켰을 때 약간 노래지는 현상

정상임. 실제 HDR10 표준을 따라가고 있다는 증거. SDR 상태에서 파랗게 보이는건 제조사에서 더 쨍해 보이게 하기 위해 한거고, 비표준임. 약간 노래지는게 표준이 맞음.

## sRGB 색상 강도

100%로 하는 게 표준임. 이 옵션이 존재하는 이유는, SDR로 만들어진 콘텐츠가 이정도로 밝은 색상을 고려하지 않았을 때, 너무 과하게 쨍해 보일 수 있는 문제를 어느정도 해결하기 위한 옵션임. 그러나, 그건 만든 사람 잘못이니까... 100%로 설정하는게 맞다.

## 게임 세팅

```bash
yay -S proton-ge-custom-bin
```

Steam 실행 옵션:

```
PROTON_ENABLE_WAYLAND=1 PROTON_ENABLE_HDR=1 gamemoderun %command%
```
